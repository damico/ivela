3. Documento de Setup para Desenvolvedores

  1. Faça a instalação, se não tiver o JDK 1.6 faço o download, no site: Getl lastes JDK
  2. Faça o Download do Netbeans 6.1, WEB & Java EE no site: Get latest NetBeans /
  3. Faça o download do maven da versão 2.0.9, no site: Get Lastest Maven
  4. Vá em Tools --> Plugins
  5. Escolha dos plugins disponíveis o maven e instale.
  6. Vá em Tools --> Options
    1. Na aba Miscellaneous clique em Maven 2, que se foi instalada corretamente deverá aparecer.
    2. Coloque o caminho do diretório onde o maven foi instalado. Ex: /home/$username/apache-maven-2.0.9.
    3. Marque a opção: “Always user external Maven for building projects.”
  7. Agora vá em Versioning --> Checkout -->
    1. Digite a URL do repositório: "http://200.19.176.65/svn/projibm
    2. Digite seu nome de usuário e sua senha e click em next
    3. Clique em Browser para escolher o folder do projeto, e clique em Ivela e depois em ok.
    4. Marque “Scan for Netbeans Projects after Checkout.”
    5. Em qualquer dos projetos clique em Project Files e abra o arquivo settings.xml
    6. Adicione o seguinte conteúdo a ele e salve:


  8. Faça um build em ivela (pom).
  9. Start Glassfish.
  10. Depois clique com a direita em “View Admin Console”
    1. Na tela que irá abrir faça o login com usuário: admin e senha:adminadmin.
    2. Dentro de Applications
    3. Depois em EJB Modules
    4. Clique em Deploy
    5. Clique em browse, e escolha no seguinte caminho: NetbeansProjects --> ivela -->ivela-ejb --> target --> ivela-ejb-1,0-with-dependencies.jar.
    6. Dê OK, pronto, agora a cada mudança feita no ejb deve ser feito um redeploy, utilizando o mesmo diretório.
  11. Para redeploy, refaça desde o passo 9.
  12. Depois dê um run em ivela-web(war).
